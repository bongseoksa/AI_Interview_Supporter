본 문서는 AI 면접 서비스의 **MVP 기준 IA(Information Architecture) 및 요구사항정의서**이다.

기획 의도는 기능 나열이 아닌 **사용자 흐름 중심 + AI 개입 지점의 명확화**에 있다.

---

## 1. IA 개요

### 1-1. IA 설계 원칙

- 면접자는 “평가 대상”이 아닌 **훈련 주체**이다.
- 모든 플로우는 **면접 시작 전 → 진행 → 종료 → 회고**의 단일 흐름을 따른다.
- AI 개입 지점은 다음 단계까지만 허용한다.
    - 질문 생성
    - 꼬리 질문 판단
    - 답변 평가
    - 피드백 생성
- 점수는 **내부 지표**로만 사용하며, UI에는 직접 노출하지 않는다.

---

## 2. 전체 IA 구조 (MVP)

```
Root
 ├─ 홈
 │   └─ 면접 시작
 ├─ 내정보
 │   └─ JD 설정
 ├─ 면접 설정
 │   ├─ 면접 유형 선택
 │   ├─ 면접 목적 선택
 │   └─ 면접관 유형 선택
 ├─ 면접 진행
 │   ├─ 질문/답변 화면
 │   └─ 음성 입력(STT)
 ├─ 면접 결과
 │   ├─ 요약 피드백
 │   └─ 상세 피드백
 └─ 면접 이력
     ├─ 이력 목록
     └─ 이력 상세

```

## 3. 화면별 요구사항 정의

---

### 3-1. 홈

**목적**

서비스 진입 및 면접 시작 진입점 제공

**기능 요구사항**

- [필수] 면접 시작 버튼 제공
- [필수] JD 미설정 시 면접 시작 버튼 비활성화
- [선택] 최근 면접 요약 정보 표시

---

### 3-2. 내정보 (JD 설정)

**목적**

면접 질문·난이도·평가 기준의 기준점 설정

**기능 요구사항**

- [필수] 직군 입력 (Select)
- [필수] 연차 입력 (Select)
- [옵션] 세부 스택 입력 (Multi Select / Text)
- [필수] 저장 버튼
- [필수] JD 미입력 시 면접 진행 불가 처리

**비고**

- JD는 모든 면접 세션에 스냅샷 형태로 저장된다.

---

### 3-3. 면접 설정

### 3-3-1. 면접 유형 선택

- 옵션
    - 1:1 (기본값)
    - 다:1
    - 다:다
- 다:다 선택 시
    - AI 면접자 자동 포함 (최대 1명)
    - AI 면접자는 동일 JD·연차 기준

---

### 3-3-2. 면접 목적 선택

- 기술 / 인성 / 상황
- 최소 1개 이상 필수
- 선택 조합에 따라 질문 비율 자동 계산

---

### 3-3-3. 면접관 유형 선택

- 기본형 (기본값)
- 압박형
- 유연형

---

### 3-4. 면접 진행 화면

**목적**

질문–답변 인터랙션에 집중된 단일 화면

**기능 요구사항**

- [필수] 텍스트 질문 출력
- [필수] 텍스트 답변 입력
- [선택] 음성 답변 입력(STT)
- [필수] 답변 시작/종료 버튼
- [필수] 실시간 음성 → 텍스트 변환 표시
- [필수] 질문 진행 상태 표시 (n / 8)
- [필수] 면접 종료 버튼

**제약사항**

- 최대 질문 수: 8문항
- 꼬리 질문: 질문당 최대 2회
- AI 면접자 답변에는 꼬리 질문 미발생

---

### 3-5. 면접 결과 화면

**목적**

점수 노출 없이 학습 중심 피드백 제공

**기능 요구사항**

- [필수] 면접 요약 피드백
- [필수] 항목별 평가 결과 (구간화)
- [필수] 개선 포인트 서술형 피드백
- [선택] 이전 면접 대비 변화 코멘트

**비노출 정책**

- 정량 점수(0~100)는 UI에 노출하지 않는다.

---

### 3-6. 면접 이력

### 3-6-1. 이력 목록

- 표시 정보
    - 면접 일시
    - 면접 유형
    - 면접 목적
    - 요약 피드백

---

### 3-6-2. 이력 상세

- 질문·답변 전체 로그
- 질문별 피드백
- 항목별 종합 피드백
- 전체 면접 요약

**내부 데이터 포함 (비노출)**

- 질문별 점수
- 항목별 점수
- 면접관 유형별 평가

---

## 4. AI 개입 지점 정리 (Boundary)

| 단계 | AI 개입 여부 |
| --- | --- |
| JD 기반 질문 생성 | O |
| 꼬리 질문 판단 | O |
| 답변 점수 산정 | O (비노출) |
| 사용자 점수 표시 | X |
| 서술형 피드백 생성 | O |

---

## 5. MVP 범위 고정 선언

- 실시간 코칭, 답변 중 개입: 제외
- 타 사용자 비교/랭킹: 제외
- 합불 판정: 제외

---

### MVP 핵심 문장

>
