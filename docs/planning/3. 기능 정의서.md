# 면접 서비스 기능정의서 (MVP)

본 문서는 **면접 서비스 MVP 범위**에 대한 기능정의서이다.

플로우차트를 기준으로 기능을 식별하고, 구현 가능한 수준으로 정의한다.

---

## 1. 공통 규칙

- JD 미설정 시 면접 관련 기능 접근 불가
- 모든 면접 데이터는 로그로 저장
- AI 평가 로직은 화면에 노출하지 않음
- 점수 범위: 0 ~ 100

---

## 2. 내정보 / JD 설정

### F-01. JD 입력

- **설명**: 면접 진행을 위한 JD 정보를 입력한다.
- **입력값**
    - 직군 (필수)
    - 연차 (필수)
    - 세부 스택 (옵셔널)
- **출력**: JD 저장 완료 상태
- **조건**: 직군, 연차 미입력 시 저장 불가
- **우선순위**: 필수

---

## 3. 면접 설정

### F-02. 면접 유형 선택

- **설명**: 면접 진행 방식을 선택한다.
- **옵션**
    - 1:1 (기본값)
    - 다:1
    - 다:다
- **제약**
    - 다:다 선택 시 AI 면접자 자동 생성
- **우선순위**: 필수

---

### F-03. 면접 목적 선택

- **설명**: 질문 유형 비율을 결정한다.
- **옵션**
    - 기술면접
    - 인성면접
    - 상황면접
- **규칙**
    - 최소 1개 이상 필수 선택
    - 선택 조합에 따라 질문 비율 자동 계산
- **우선순위**: 필수

---

### F-04. 면접관 유형 선택

- **설명**: 면접관의 질문 스타일을 설정한다.
- **옵션**
    - 기본형 (디폴트)
    - 압박형
    - 유연형
- **우선순위**: 필수

---

## 4. 면접 진행

### F-05. 면접 시작

- **설명**: 설정된 정보로 면접을 시작한다.
- **선행조건**
    - JD 입력 완료
    - 면접 설정 완료
- **출력**: 첫 질문 생성

---

### F-06. 질문 생성

- **설명**: JD와 면접 목적에 따라 질문을 생성한다.
- **로직**
    - JD 기반 질문 주제 선정
    - 목적 비율에 따른 질문 분배
    - 반구조화 질문 방식 적용
- **최대 질문 수**: 8문항

---

### F-07. 면접자 답변 입력

- **설명**: 면접자가 질문에 답변한다.
- **방식**
    - 텍스트 입력
    - 음성 입력(STT)
- **STT 규칙**
    - 시작 / 종료 버튼 기반
    - 실시간 텍스트 변환 표시
- **우선순위**: 필수

---

### F-08. 꼬리질문 생성

- **설명**: 답변 기반 추가 질문을 생성한다.
- **조건**
    - 질문 의도에 부합한 답변
    - 추가 탐색 포인트 존재
- **제약**
    - 질문당 최대 2회
    - AI 면접자 답변에는 적용하지 않음

---

### F-09. 다:다 면접 처리

- **설명**: 다:다 면접에서 AI 면접자 로직을 처리한다.
- **규칙**
    - AI 면접자는 평가 대상 아님
    - AI 면접자 답변은 질문 수 미포함
    - AI 면접자 답변 후 면접자에게 답변 요청

---

## 5. 평가 및 종료

### F-10. 답변 평가

- **설명**: 각 답변을 점수와 근거로 평가한다.
- **평가 항목**
    - 질문별 점수
    - 항목별 점수
- **비고**: 화면 비노출, 이력용 저장

---

### F-11. 면접 종료

- **설명**: 면접을 종료한다.
- **조건**
    - 사용자 종료 버튼 클릭
    - 최대 질문 수 도달

---

### F-12. 면접 결과 생성

- **설명**: 면접 결과를 종합한다.
- **포함 정보**
    - 질문별 점수 및 피드백
    - 항목별 점수 및 피드백
    - 총점 (목적 비율 반영)
    - 종합 피드백

---

## 6. 면접 이력 관리

### F-13. 면접 이력 저장

- **설명**: 면접 결과를 단건 이력으로 저장한다.
- **저장 항목**
    - 시작/종료 시간
    - JD 정보
    - 면접 유형
    - 면접 목적
    - 질의응답
    - 평가 결과

---

### F-14. 면접 이력 조회

- **설명**: 이전 면접 이력을 조회한다.
- **기능**
    - 이력 목록 조회
    - 요약 정보 표시
    - 상세 이력 조회

---

## 7. 스펙 아웃 (MVP 제외)

- 관리자 페이지
- 이력서 업로드 기반 질문 생성
- TTS 기반 음성 질문
